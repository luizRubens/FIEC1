Logica de Programação Orientada a Objetos
  • Lógica de programação é uma materia comumente encontrada nos cursos de informatica e tem como função ajudar o aluno a desenvolver os primeros conceitos e sintaxes de como funciona a programação utilizando como base uma linguagem de programação. Essa materia tambem tem como função ensinar os primeiros passos para a progrmação utlizando metodos padrões parecidos com receitas de bolo.
  • Na receita, devem-se seguir os passos para o bolo ficar pronto e sem nenhum problema. Na informática, os programadores escrevem as “receitas de bolo” (algoritmos) de modo que o computador leia e entenda o que deve ser feito, ao executar o algoritmo. Para isto é necessário uma .
  • A linguagem de programação é como uma língua normal, um grupo de palavras com significados. No caso da programação, a maioria das linguagens é escrita em Inglês. Estas linguagens fazem o computador assimilar cada comando e função de um algoritmo, depois executar cada função.
  • A linguagem de programação é somente como se escreve o algoritmo. O grande problema para muitos é o que “dizer” para o computador fazer o que é desejado.
